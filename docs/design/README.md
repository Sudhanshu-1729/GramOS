# UI/UX Design System Guidelines - RuralOS

## 1. Aesthetic Philosophy
RuralOS takes design inspiration from top-tier modern fintech platforms (**Stripe**, **CRED**, and **Mercury**) to make alternative underwriting feel like a premium, enterprise-grade national product.
- **Background**: Deep Slate / Pitch-Black `#020202` for high contrast.
- **Panels**: Glassmorphic card grids with 1px borders (`border-white/[0.04]`).
- **Typography**: `Inter` font for text density, `Outfit` font for bold displays.

---

## 2. Palette Tokens Configuration
* **Primary (Emerald Green)**: Representing healthy credit parameters, approvals, and normal climate greenness.
* **Secondary (Teal)**: Representing stable transaction cash flows and system online status.
* **Accent (Gold)**: Used for high-tier appraisal scores (DNA Platinum Card) and key actions.
* **Risk States**:
  * **Low Risk**: Emerald Green `#10b981`
  * **Medium Risk**: Gold/Amber `#f59e0b`
  * **High Risk**: Rose/Red `#ef4444`

---

## 3. Keyboard Shortcut Map (Linear Style)
To navigate the entrepreneur dashboard rapidly, the root panel binds the following key down listeners:
- **`[D]`**: Dashboard View Tab
- **`[T]`**: Digital twin visual relationship node tab
- **`[S]`**: What-If Simulator stress test panel
- **`[G]`**: Govt Schemes catalog
When activated, a custom bottom-right feedback toast displays navigation updates.
